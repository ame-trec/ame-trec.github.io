# 作りたい車両を選定しよう
リテクスチャにも慣れてきたら、いよいよ自炊に足を踏み入れてみましょう。  
まず最初に、作りたい車両を決めましょう。自分の好きな車両、地元の鉄道の車両だと作りやすいかと思います。  
ただし…
![西武001系](./seibu.JPG)
このような丸みを帯びた車両や新幹線や特急のような流線型の車両、複雑な造形の車両は**初心者のうちはまず避けたほうが無難です**。  
造形で頭を悩ませ、**絶対挫折します**。 慣れてからのほうが良いでしょう。 
![筑豊電鉄](./chikutetsu.JPG)
また、ドアが折戸の車両も避けておくといいです。MTRでは折戸を再現するのが困難だからです。  
今回は作例として、東急9000系を作ります。    
# 資料集めをしよう
では早速資料集めをしていきましょう。ネットで集めても良いですし、実際に乗ってみて集めるのもいいですね。  
ただし、ネットで集める際には資料写真は配布データに含めないようにしましょう。  
資料集めに関しては、[こちらのサイト](https://myzk-traintech.jimdoweb.com/railsim/pi%E5%88%B6%E4%BD%9C%E8%AC%9B%E5%BA%A7/%E3%81%9D%E3%81%AE2-%E8%B3%87%E6%96%99%E5%8F%8E%E9%9B%86%E7%B7%A8/)が参考になります。  
  
次はいよいよ自炊です。
